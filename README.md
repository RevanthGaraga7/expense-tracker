# Expense Tracker 💰

A simple **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript** that allows users to record income and expenses and automatically calculate the balance.

🌐 **Live Demo:**  
https://revanthgaraga7.github.io/expense-tracker/

---

# 📌 Features

- Add income and expense transactions
- Automatic balance calculation
- Shows total credited and deducted amounts
- Transaction history modal
- Reset all transactions
- Clean responsive UI

---

# 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage for saving transactions

---

# 📷 Application Preview

Expense Tracker dashboard showing:

- Current Balance

- Amount Deducted
- Transaction list
- History modal
